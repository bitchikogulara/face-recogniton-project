# Internship Topic: Development of a Facial Recognition Mobile Application and Integration with Connected Devices

## Background

With the rapid advancement of facial recognition technologies and the growing adoption of connected devices (IoT), innovative ways of interacting with our environment have become possible. 
The objective of this internship is to develop a mobile application capable of recognizing faces through a smartphone camera and communicating with various connected devices based on the recognition results.

## Objectives

### 1. Research and State of the Art

- Study the different facial recognition technologies available for mobile devices, including frameworks and libraries (such as OpenCV and TensorFlow Lite).
- Analyze communication protocols commonly used for connected devices (including Bluetooth, Wi-Fi, MQTT, and other IoT communication standards).
- Evaluate the advantages, limitations, and performance of existing solutions for real-time facial recognition on smartphones.

### 2. Mobile Application Development

- Design and develop a mobile application for iOS and/or Android platforms incorporating facial recognition capabilities.
- Implement real-time face detection and face recognition algorithms.
- Develop an intuitive and user-friendly interface for managing recognized users and connected devices.
- Test and optimize the application to ensure efficient performance across different smartphone models and hardware configurations.
- Ensure low latency, high recognition accuracy, and efficient resource utilization.

### 3. Integration with Connected Devices

- Identify and select various types of connected devices, such as smart lights, smart locks, voice assistants, and other IoT-enabled systems.
- Optionally design and manufacture a custom connected device using the laboratory's 3D printing facilities.
- Integrate electronic components and communication modules to enable Bluetooth connectivity for the custom device.
- Develop interaction scenarios based on facial recognition outcomes. Examples include:
  - Automatically turning on lights when an authorized user is recognized.
  - Unlocking a smart lock upon successful user authentication.
  - Triggering a voice alarm or notification when an unauthorized individual is detected.
  - Executing personalized actions depending on the recognized user's identity.

### 4. Testing and Validation

- Conduct user testing to evaluate the accuracy, reliability, and responsiveness of the facial recognition system.
- Assess the system's robustness against spoofing attempts and determine whether the recognition mechanism can be deceived using photographs, videos, or other attack methods.
- Verify seamless communication and interaction with connected devices.
- Collect user feedback and performance metrics to identify potential improvements.
- Analyze system behavior under different environmental conditions, such as varying lighting levels, facial expressions, and camera angles.

### 5. Documentation and Final Presentation

- Prepare comprehensive technical documentation describing:
  - System architecture
  - Design decisions
  - Development methodology
  - Implemented technologies
  - Testing procedures and results
- Document challenges encountered during development and the solutions implemented.
- Prepare and deliver a final presentation showcasing:
  - Project objectives
  - Technical implementation
  - Demonstration of the application and connected devices
  - Experimental results and performance evaluation
  - Future improvements and potential applications

## Expected Outcomes

By the end of the internship, the project should deliver:

- A fully functional mobile application capable of real-time facial recognition.
- Integration with one or more connected devices through Bluetooth, Wi-Fi, or MQTT communication.
- Demonstration of intelligent interactions based on user identification.
- Performance evaluation and security analysis of the facial recognition system.
- Complete technical documentation and a final project presentation.
